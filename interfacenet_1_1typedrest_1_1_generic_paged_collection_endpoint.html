Detailed Description

REST endpoint that represents a collection of TEntitys as TElementEndpoints with pagination support.

Use the more constrained PagedCollectionEndpoint when possible.

Parameters
<TEntity>The type of entity the endpoint represents.
<TElementEndpoint>The specific type of ElementEndpoint to provide for individual TEntitys.
Deprecated:
Use CollectionEndpoint instead.
