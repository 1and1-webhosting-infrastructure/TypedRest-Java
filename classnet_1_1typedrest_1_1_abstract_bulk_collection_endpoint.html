Detailed Description

Base class for building REST endpoints that represents a collection of TEntitys as TElementEndpoints with bulk create and replace support.

Parameters
<TEntity>The type of entity the endpoint represents.
<TElementEndpoint>The specific type of ElementEndpoint to provide for individual TEntitys.
Deprecated:
Use AbstractCollectionEndpoint instead.
