Detailed Description

Base class for building view components operating on a GenericStreamEndpoint.

To enable server-push (real-time actualization of the web-ui), please make sure to have your UI annotated with Push.

Parameters
<TEntity>The type of entity the TEndpoint represents.
<TEndpoint>The specific type of GenericStreamEndpoint to operate on.
<TElementEndpoint>The specific type of ElementEndpoint the TEndpoint provides for individual TEntitys.
