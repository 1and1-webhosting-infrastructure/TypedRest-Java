Detailed Description

Component for creating a new element in a GenericCollectionEndpoint.

Use the more constrained CreateElementView when possible.

Parameters
<TEntity>The type of entity to create.
<TElementEndpoint>The specific type of ElementEndpoint the CollectionEndpoint provides for individual TEntitys.
