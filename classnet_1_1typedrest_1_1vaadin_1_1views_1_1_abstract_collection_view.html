Detailed Description

Base class for building view components operating on an CollectionEndpoint.

Parameters
<TEntity>The type of entity the TEndpoint represents.
<TEndpoint>The specific type of CollectionEndpoint to operate on.
<TElementEndpoint>The specific type of ElementEndpoint the TEndpoint provides for individual TEntitys.
